# PartyKeys Lessons · 产品介绍

> **音乐密码 · 多邻国式 MIDI 乐理学习系统**
> 上次更新：2026-05-04 · 版本 v0.7.3

---

## 1. 一句话定位

> **"插上 PartyKeys，跟着灯走，5 分钟一课，零基础也能在 4 周内会弹《Kiss the Rain》。"**

把 Duolingo 的关卡机制 × PartyKeys 36 键 LED 钢琴 × 音乐密码主线课程内容，融合成一个浏览器单页应用，让零基础用户在游戏化体验中完成 80+ 节系统乐理 + 钢琴弹奏教学。

---

## 2. 核心价值

| 痛点 | PartyKeys Lessons 的解法 |
|------|-------------------------|
| 传统琴课贵、慢、不方便 | 浏览器打开就用，5 分钟一关，完全自主进度 |
| 五线谱 / 简谱 / 西式音名 让初学者懵 | 双包并行：基础课用 CDEFGAB，进阶用简谱 1234567，用户自选 |
| 看不到"自己该弹哪" | LED 实时高亮目标键，错音红色反馈，3 秒钟看懂"该按这里" |
| 学完忘、没成就感 | Duolingo 全套激励：3 颗心 / Streak / XP 等级 / 1-3 星 / 撒花动画 |
| 单调的练习曲枯燥 | 用真实歌曲（欢乐颂、雨的印记、孤勇者、月亮代表我的心 …）做练习内容 |

## 3. 产品结构

应用顶部用 **大 Tab 切换** 两个独立的"卡包"，存档独立、UI 独立、机制相同：

### 🟢 基础课《Music Code Duolingo Basics》— 7 关
- 西式音名 CDEFGAB
- LED 角色色（绿=该弹、红=错了、紫=和弦根音 …）
- 适合：完全没接触过任何乐理的成人 / 儿童入门
- 内容：MIDI 设备识别 → CDE 三兄弟 → 七个白键 → 高八度概念 → 弹《小星星》
- 完成后自动推荐进入"进阶包"

### 🟣 进阶包《Music Code Duolingo Advanced》— 80 关
- 中国简谱 1234567 + 上下点八度
- 音乐密码彩虹色（Do=红 Re=橙 Mi=黄 Fa=绿 Sol=青 La=蓝 Si=紫）
- 4 章 × 5 节 × 3 玩法（跟弹/连弹/挑战）的完整 4-周课程体系

| 章 | 主题 | 关数 | 代表曲目 |
|------|------|------|---------|
| 第一章 | 基础入门 | 17 | 失恋阵线联盟 / 欢乐颂 / 王妃 / 粉刷匠 / 听妈妈的话 / 依然爱你 / 不能说的秘密 / 丢手绢 / 宝贝 / 孤勇者 |
| 第二章 | 指法技巧 | 23 | 简谱认识 + 把位移动 + 扩指 + **完整《雨的印记》第 1-4 段 + 整曲 boss** |
| 第三章 | 和弦弹唱 | 13 | 6 个基础和弦 + 1645 / 6415 / 4536251 / 15634125 走向（春天里 / 杀破狼 / 寂寞沙洲冷 / 时间都去哪了 / 大城小爱 / 遇见 / 听海） |
| 第四章 | 双手合奏 | 21 | 弹唱谱根音 + 综合走向 + **毕业 boss：完整双手版《Kiss the Rain》80 XP** |

## 4. 全套激励机制（Duolingo 同款）

| 机制 | 实现 |
|------|------|
| **3 颗爱心 ❤️** | 错音扣心，扣完出关，每 15 分钟回 1 颗（成人模式严扣，儿童模式自动演示，不扣心） |
| **🔥 Streak 连续天数** | 全包共享，每天打卡至少完成 1 关，连续 7 天奖励徽章 |
| **✦ XP / 等级** | 每关 +10-20 XP，boss 关 +30-80 XP，每 100 XP 升 1 级，10 个等级称号 |
| **⭐⭐⭐ 三颗星** | 完成 = 1 ⭐，0-1 错 = 2 ⭐，零错零提示 = 3 ⭐，3 ⭐ 触发撒花动画 |
| **路径地图** | Duolingo 经典 S 弯小路，已通的发亮，下一关呼吸光晕，未解锁灰色 + 锁图标 |
| **关卡式解锁** | 严格顺序解锁，过一关开下一关 |

## 5. 8 大题型组件

| 题型 | 名称 | 玩法 | LED 角色 |
|---|------|------|---------|
| **T1** | 照亮弹 | LED 亮一个键，用户照着弹 | 主教学，告诉你弹哪 |
| **T2** | 听后弹 | 系统播放一个音，用户复现 | 隐藏，只在提示时短亮 |
| **T3** | 听辨四选一 | 听一段，从 4 个文字选项中点选 | 不参与 |
| **T7** | 序列复现 | LED 依次亮 N 个键，全亮完用户按顺序弹回 | 逐键提示 → 隐藏复现 |
| **T8** | 小演奏 | 完整弹一段 8-16 小节，节拍器跟随，LED 提示难句 | 半提示，副歌亮、主歌暗 |
| **T_ANY** | 自由弹奏 | 任意弹 N 个键热身 | 全键扫一次 |
| T4/T5/T6 | 找错音/填空/节奏拍 | v2 路线图（已设计未实装） | — |

## 6. LED 协议精确对齐

屏幕亮键颜色严格匹配 PartyKeys 36 键真实 LED 协议色（CLAUDE.md 规范）：

| 角色 | LED 协议码 | 实际色值 | 用途 |
|---|---|---|---|
| 弹这里 / 对了 | 0x07 green | `#0e7f1f` | 主目标提示 |
| 候选 / 备选 | 0x05 yellow | `#7f5500` | T5 填空 |
| 节拍位置 | 0x09 blue | `#08307f` | 节奏类 |
| 和弦根音 | 0x0C purple | `#4b007f` | 和弦模块 |
| 和弦三音 | 0x03 orange | `#7f1500` | 和弦模块 |
| 参考音 | 0x08 cyan | `#137f60` | T2/T4 |
| 弹错了 | 0x01 red | `#7f0000` | 全局错误反馈 |

进阶包额外切换"音乐密码彩虹色"模式：每个音名固定一个色，不再按角色切换。Do = 红 / Re = 橙 / Mi = 黄 / Fa = 绿 / Sol = 青 / La = 蓝 / Si = 紫，让用户一边弹一边形成"色 = 音"的肌肉记忆。

## 7. 体验细节亮点

### 双语 + 双语气
- 顶栏右上一键 EN / 中文 切换，全站 89 个 DOM 标签即时刷新
- 🧒 儿童模式 / 🧑 成人模式 切换：**同一关不同文案语气**
  - 同样讲"大三和弦"，儿童："把三只小猫排成 1、3、5！🐱🏠"
  - 成人："大三和弦 = 根音 + 大三度 + 纯五度。"

### 错题处理
- **儿童模式**：错 3 次自动 LED 演示 + 鸣鸣讲解，不扣心，绝不卡住
- **成人模式**：错 1 次扣 1 心，3 心扣完重玩

### 见缝插针的小 tips
T1/T2 题型自动根据目标音生成手指提示：
- 弹 Do → 💡 拇指弹 Do
- 弹 Mi → 💡 中指弹 Mi
- 弹 Sol → 💡 小指弹 Sol
- 高音区自动加"（高音）"，低音区"（低音）"

### MIDI 快捷键
完成一题后，**直接弹高音 Do（C5）就能跳下一题**，不用回去碰鼠标。结算页同样：弹 Do = 回路径地图。一气呵成。

### 鸣鸣 3D 吉祥物
PartyKeys 紫色耳机小鹦鹉（鸣鸣 Ming-Ming）以 3D 模型形式出现在课前 / 错题 / 结算页，4 种 mood：
- 🤩 happy（开心自转）
- 🤔 scratch（挠头）
- 😴 sleep
- 💃 dance（满分时高速旋转撒花）

### 背景音乐
顶栏右上音乐按钮一键开关，开启后听到柔和的 4 和弦循环钢琴 pad（C → Am → F → G），32 秒一圈无限循环，主音量 0.04，"弱弱淡淡"不抢戏。

## 8. 技术架构

### 单文件 SPA · 零依赖
- 全部 HTML + CSS + JS + SVG 在一个 `index.html`（103 KB）
- 唯一外部资源：`avatar.glb`（266 KB，gltf-transform 压缩自 20 MB 原始模型）
- CDN 依赖：Google Fonts（Plus Jakarta Sans + DM Mono）+ `<model-viewer>`（仅 3D 鸣鸣）
- 无需构建、无需 npm、双击即可在 Chrome 打开

### MIDI / LED 引擎
- **MIDI 输入**：Web MIDI API，过滤 `/partykey/i` 设备名
- **LED 输出**：SysEx 协议（需固件 v007+），全速点亮、全关闭、单灯开关
  - 初始化：`F0 05 30 7F 7F 20 00 0F 05 F7`
  - 点亮：`F0 05 30 7F 7F 20 00 71 [count] [note color]... F7`
- **音频**：Web Audio API 合成钢琴（4 谐波 + ADSR 包络），velocity 力度感应
- **键盘**：PartyKeys 36 键，MIDI 48-83（C3-B5），21 白 + 15 黑，绝对定位精确居中

### 持久化
- `localStorage` 存档，key `pk_lessons_v1`
- 分包独立：`state.progress.basic.stars` / `state.progress.advanced.stars`
- 自动迁移老版本扁平结构（v0.6 之前的存档不丢）
- 状态：xp、level、hearts、streak、starsByLesson、unlockedUpTo、lang、tone、bgm、activePack

## 13. 与 PartyKeys Arcade 的关系

PartyKeys Arcade（game.partykeys.org）是已上线的 16 款 MIDI 游戏合集，定位"玩耍 / 娱乐"。
PartyKeys Lessons（lessons.partykeys.org）是新建的乐理学习系统，定位"学习 / 进阶"。

```
零基础用户 ──► Lessons（学）──► Arcade（玩）
                  │                       │
                  │                       │
            完成基础 / 进阶包 后        每个游戏对应一种音乐能力
                  │                       │
                  ▼                       ▼
          [基础毕业]               [Pitch Perfect]（旋律毕业推荐）
          [进阶毕业]               [Chord Detective]（和弦毕业推荐）
                                   [Highway Hero]（节奏毕业推荐）
                                   [Improv Island]（综合毕业推荐）
```

两站共用 PartyKeys 36 键硬件 + SysEx LED 协议 + 紫色品牌色，但视觉气质不同：
- Arcade：深紫 + 霓虹，强调"游戏感"
- Lessons：暖米 + Duolingo 草绿，强调"学习感"

形成完整的"**学 → 玩 → 创作**"闭环。

## 14. 快速开始（用户视角）

1. 把 PartyKeys 36 键 USB 插到 Mac/Windows/Android
2. 浏览器打开 `https://lessons.partykeys.org`（推荐 Chrome）
3. 顶栏自动检测设备，显示绿色"PartyKeys 已连接"
4. 选择卡包 → 进入第一关 → 跟着 LED 灯走
5. 每关 5 分钟，弹错了灯红，弹对了灯绿，越弹越熟
